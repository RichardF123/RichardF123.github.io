Pandas 入门笔记

Pandas 是什么

Pandas 是 Python 中最常用的数据分析库之一。它适合处理表格数据，例如 CSV、Excel、实验记录、Kaggle 数据集和机器学习训练数据。

如果说 NumPy 更像“高效数组计算工具”，那么 Pandas 更像“带行列标签的表格处理工具”。

核心数据结构

Pandas 里最重要的是两个对象：

  • Series：一列数据，类似带索引的一维数组。
  • DataFrame：二维表格，类似 Excel 表。

和机器学习的关系

Pandas 经常用在机器学习流程的前半段：

  1. 读取数据。
  2. 查看缺失值和异常值。
  3. 清洗文本或数值。
  4. 构造特征。
  5. 划分训练集和测试集。
  6. 保存预测结果。
