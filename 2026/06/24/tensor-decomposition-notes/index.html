Tensor 分解入门笔记：从基本记号到 CP、Tucker 与 PARAFAC2

这篇笔记主要根据 Kolda 与 Bader 的综述论文 Tensor Decompositions and Applications 整理，用来建立张量分解的基础概念。可以把它理解成从矩阵分解继续往高维数组推广：向量是一阶张量，矩阵是二阶张量，三维及以上数组就是高阶张量。

1. 基本记号

张量的 order，也叫 way 或 mode，指的是张量的维度个数。

  • 标量是 0 阶张量，通常用小写字母表示，例如 aaa。
  • 向量是 1 阶张量，通常用粗体小写字母表示，例如 a\mathbf{a}a。
  • 矩阵是 2 阶张量，通常用粗体大写字母表示，例如 A\mathbf{A}A。
  • 三阶及以上的张量通常用花体字母表示，例如 X\mathcal{X}X。

一些常见索引写法：

  • 向量 a\mathbf{a}a 的第 iii 个元素写作 aia_iai​。
  • 矩阵 A\mathbf{A}A 的第 (i,j)(i,j)(i,j) 个元素写作 aija_{ij}aij​。
  • 三阶张量 X\mathcal{X}X 的第 (i,j,k)(i,j,k)(i,j,k) 个元素写作 xijkx_{ijk}xijk​。
  • NNN 阶张量 X\mathcal{X}X 的元素可以写作 xi1i2⋯iNx_{i_1 i_2 \cdots i_N}xi1​i2​⋯iN​​。

如果 X∈RI1×I2×⋯×IN\mathcal{X}\in \mathbb{R}^{I_1\times I_2\times\cdots\times I_N}X∈RI1​×I2​×⋯×IN​，一般约定第 nnn 个 mode 的索引范围是 in=1,…,Ini_n=1,\ldots,I_nin​=1,…,In​。

2. Fiber 与 Slice

张量里的子结构很重要，因为很多张量运算其实就是在某个 mode 上对 fiber 或 slice 做矩阵操作。

Fiber 可以理解成“张量中的一根线”。当固定所有索引，只保留一个索引变化时，得到的就是 fiber。

  • 对矩阵来说，列向量和行向量就是 mode-1 fiber 和 mode-2 fiber。
  • 对三阶张量来说，有三类 fiber：
    • mode-1 column fiber：X:jk\mathcal{X}_{:jk}X:jk​
    • mode-2 row fiber：Xi:k\mathcal{X}_{i:k}Xi:k​
    • mode-3 tube fiber：Xij:\mathcal{X}_{ij:}Xij:​

Slice 可以理解成“张量中的一个切片”。当固定除两个索引外的其他索引时，得到的就是 slice。

对三阶张量来说，有三类 slice：

  • horizontal slice：Xi::\mathcal{X}_{i::}Xi::​
  • lateral slice：X:j:\mathcal{X}_{:j:}X:j:​
  • frontal slice：X::k\mathcal{X}_{::k}X::k​，也常简写为 Xk\mathbf{X}_kXk​

3. 范数与内积

张量的 Frobenius 范数可以看成矩阵 Frobenius 范数的高维推广。对

X∈RI1×I2×⋯×IN\mathcal{X}\in \mathbb{R}^{I_1\times I_2\times\cdots\times I_N} X∈RI1​×I2​×⋯×IN​

它的范数定义为：

∥X∥=∑i1=1I1∑i2=1I2⋯∑iN=1INxi1i2⋯iN2\|\mathcal{X}\| = \sqrt{ \sum_{i_1=1}^{I_1} \sum_{i_2=1}^{I_2} \cdots \sum_{i_N=1}^{I_N} x_{i_1i_2\cdots i_N}^{2} } ∥X∥=i1​=1∑I1​​i2​=1∑I2​​⋯iN​=1∑IN​​xi1​i2​⋯iN​2​​

两个同尺寸张量 X\mathcal{X}X 与 Y\mathcal{Y}Y 的内积定义为对应位置元素乘积之和：

⟨X,Y⟩=∑i1=1I1∑i2=1I2⋯∑iN=1INxi1i2⋯iNyi1i2⋯iN\langle \mathcal{X},\mathcal{Y}\rangle = \sum_{i_1=1}^{I_1} \sum_{i_2=1}^{I_2} \cdots \sum_{i_N=1}^{I_N} x_{i_1i_2\cdots i_N}y_{i_1i_2\cdots i_N} ⟨X,Y⟩=i1​=1∑I1​​i2​=1∑I2​​⋯iN​=1∑IN​​xi1​i2​⋯iN​​yi1​i2​⋯iN​​

4. Rank-One Tensor

一个 NNN 阶张量如果可以写成 NNN 个向量的外积，就称为 rank-one tensor：

X=a(1)∘a(2)∘⋯∘a(N)\mathcal{X} = \mathbf{a}^{(1)}\circ \mathbf{a}^{(2)}\circ \cdots \circ \mathbf{a}^{(N)} X=a(1)∘a(2)∘⋯∘a(N)

其中 ∘\circ∘ 表示 outer product。元素级别写法是：

xi1i2⋯iN=ai1(1)ai2(2)⋯aiN(N)x_{i_1i_2\cdots i_N} = a^{(1)}_{i_1} a^{(2)}_{i_2} \cdots a^{(N)}_{i_N} xi1​i2​⋯iN​​=ai1​(1)​ai2​(2)​⋯aiN​(N)​

三阶 rank-one tensor 可以理解成三个向量 a,b,c\mathbf{a},\mathbf{b},\mathbf{c}a,b,c 的外积：

X=a∘b∘c\mathcal{X}=\mathbf{a}\circ \mathbf{b}\circ \mathbf{c} X=a∘b∘c

这个概念是 CP 分解的核心，因为 CP 分解就是把一个复杂张量写成多个 rank-one tensor 的和。

5. 对称张量与对角张量

如果一个张量每个 mode 的尺寸都相同，例如

X∈RI×I×⋯×I\mathcal{X}\in \mathbb{R}^{I\times I\times\cdots\times I} X∈RI×I×⋯×I

它叫 cubical tensor。

如果在任意索引置换下元素值都不变，那么它是 supersymmetric tensor。例如三阶张量满足：

xijk=xikj=xjik=xjki=xkij=xkjix_{ijk}=x_{ikj}=x_{jik}=x_{jki}=x_{kij}=x_{kji} xijk​=xikj​=xjik​=xjki​=xkij​=xkji​

如果只在部分 mode 上对称，就叫 partially symmetric tensor。例如

X∈RI×I×K\mathcal{X}\in \mathbb{R}^{I\times I\times K} X∈RI×I×K

如果每个 frontal slice 都是对称矩阵，即

Xk=XkT\mathbf{X}_k=\mathbf{X}_k^T Xk​=XkT​

那么它就在前两个 mode 上部分对称。

对角张量类似矩阵里的对角矩阵。只有当

i1=i2=⋯=iNi_1=i_2=\cdots=i_N i1​=i2​=⋯=iN​

时，元素 xi1i2⋯iNx_{i_1i_2\cdots i_N}xi1​i2​⋯iN​​ 才可能非零。

6. Vectorization 与 Matricization

Vectorization 是把一个张量按固定顺序重新排列成向量，记作 vec(X)\mathrm{vec}(\mathcal{X})vec(X)。

如果 A∈Rn1×n2×⋯×nd\mathcal{A}\in\mathbb{R}^{n_1\times n_2\times\cdots\times n_d}A∈Rn1​×n2​×⋯×nd​，一个常用的列优先位置映射是：

col(i,n)=i1+(i2−1)n1+(i3−1)n1n2+⋯+(id−1)n1n2⋯nd−1\mathrm{col}(\mathbf{i},\mathbf{n}) = i_1 +(i_2-1)n_1 +(i_3-1)n_1n_2 +\cdots +(i_d-1)n_1n_2\cdots n_{d-1} col(i,n)=i1​+(i2​−1)n1​+(i3​−1)n1​n2​+⋯+(id​−1)n1​n2​⋯nd−1​

这里 i=[i1,…,id]\mathbf{i}=[i_1,\ldots,i_d]i=[i1​,…,id​] 是元素坐标，n=[n1,…,nd]\mathbf{n}=[n_1,\ldots,n_d]n=[n1​,…,nd​] 是每个 mode 的尺寸。

Matricization 也叫 unfolding 或 flattening，是把张量按某个 mode 展开成矩阵。mode-nnn matricization 记作 X(n)\mathbf{X}_{(n)}X(n)​，它会把 mode-nnn fibers 排成矩阵的列。

对张量分解来说，matricization 很重要，因为许多更新公式都可以转化成矩阵最小二乘问题。

7. n-Mode Product

mode-nnn product 表示沿着第 nnn 个 mode 用矩阵或向量去乘张量。

设

X∈RI1×I2×⋯×IN,U∈RJ×In\mathcal{X}\in \mathbb{R}^{I_1\times I_2\times\cdots\times I_N}, \quad \mathbf{U}\in\mathbb{R}^{J\times I_n} X∈RI1​×I2​×⋯×IN​,U∈RJ×In​

则 mode-nnn 矩阵乘法写作：

X×nU\mathcal{X}\times_n \mathbf{U} X×n​U

新张量的尺寸为：

I1×⋯×In−1×J×In+1×⋯×INI_1\times\cdots\times I_{n-1} \times J \times I_{n+1}\times\cdots\times I_N I1​×⋯×In−1​×J×In+1​×⋯×IN​

元素级别写法是：

(X×nU)i1⋯in−1jin+1⋯iN=∑in=1Inxi1⋯in⋯iNujin(\mathcal{X}\times_n \mathbf{U})_{i_1\cdots i_{n-1}j i_{n+1}\cdots i_N} = \sum_{i_n=1}^{I_n} x_{i_1\cdots i_n\cdots i_N}u_{j i_n} (X×n​U)i1​⋯in−1​jin+1​⋯iN​​=in​=1∑In​​xi1​⋯in​⋯iN​​ujin​​

直观理解：矩阵 U\mathbf{U}U 会对 mode-nnn fibers 做线性变换。

常用性质：

X×mA×nB=X×nB×mA,m≠n\mathcal{X}\times_m \mathbf{A}\times_n \mathbf{B} = \mathcal{X}\times_n \mathbf{B}\times_m \mathbf{A}, \quad m\ne n X×m​A×n​B=X×n​B×m​A,m=n

X×nA×nB=X×n(BA)\mathcal{X}\times_n \mathbf{A}\times_n \mathbf{B} = \mathcal{X}\times_n(\mathbf{B}\mathbf{A}) X×n​A×n​B=X×n​(BA)

如果乘的是向量 v∈RIn\mathbf{v}\in\mathbb{R}^{I_n}v∈RIn​，结果会降低一阶，因为第 nnn 个 mode 被收缩掉。

8. Kronecker、Khatri-Rao 与 Hadamard 乘积

张量分解公式里经常出现三种矩阵乘积。

Kronecker product：

A⊗B\mathbf{A}\otimes \mathbf{B} A⊗B

如果 A∈RI×J\mathbf{A}\in\mathbb{R}^{I\times J}A∈RI×J，B∈RK×L\mathbf{B}\in\mathbb{R}^{K\times L}B∈RK×L，则结果尺寸是 (IK)×(JL)(IK)\times(JL)(IK)×(JL)。

Khatri-Rao product 是按列做 Kronecker product：

A⊙B=[a1⊗b1,a2⊗b2,…,aR⊗bR]\mathbf{A}\odot \mathbf{B} = [ \mathbf{a}_1\otimes\mathbf{b}_1, \mathbf{a}_2\otimes\mathbf{b}_2, \ldots, \mathbf{a}_R\otimes\mathbf{b}_R ] A⊙B=[a1​⊗b1​,a2​⊗b2​,…,aR​⊗bR​]

它在 CP 分解的 matricized 形式中非常常见。

Hadamard product 是对应位置逐元素相乘：

(A∗B)ij=aijbij(\mathbf{A}*\mathbf{B})_{ij}=a_{ij}b_{ij} (A∗B)ij​=aij​bij​

几个常用性质：

(A⊗B)T=AT⊗BT(\mathbf{A}\otimes \mathbf{B})^T = \mathbf{A}^T\otimes \mathbf{B}^T (A⊗B)T=AT⊗BT

(A⊗C)(B⊗D)=AB⊗CD(\mathbf{A}\otimes \mathbf{C})(\mathbf{B}\otimes \mathbf{D}) = \mathbf{A}\mathbf{B}\otimes \mathbf{C}\mathbf{D} (A⊗C)(B⊗D)=AB⊗CD

(A⊙B)T(A⊙B)=(ATA)∗(BTB)(\mathbf{A}\odot \mathbf{B})^T(\mathbf{A}\odot \mathbf{B}) = (\mathbf{A}^T\mathbf{A})*(\mathbf{B}^T\mathbf{B}) (A⊙B)T(A⊙B)=(ATA)∗(BTB)

9. SVD 与 Moore-Penrose 伪逆

不是所有矩阵都能做特征值分解，但任意矩阵都可以做奇异值分解。对

A∈Rm×n\mathbf{A}\in\mathbb{R}^{m\times n} A∈Rm×n

有：

A=UΣVT\mathbf{A}=\mathbf{U}\mathbf{\Sigma}\mathbf{V}^T A=UΣVT

其中 U\mathbf{U}U 和 V\mathbf{V}V 是正交矩阵，Σ\mathbf{\Sigma}Σ 是对角矩阵，对角线上的 σ1,σ2,…\sigma_1,\sigma_2,\ldotsσ1​,σ2​,… 是奇异值。

当线性方程

Ax=b\mathbf{A}\mathbf{x}=\mathbf{b} Ax=b

中的 A\mathbf{A}A 不可逆时，可以用 Moore-Penrose 伪逆：

x=A+b\mathbf{x}=\mathbf{A}^{+}\mathbf{b} x=A+b

如果 A=UΣVT\mathbf{A}=\mathbf{U}\mathbf{\Sigma}\mathbf{V}^TA=UΣVT，则：

A+=VΣ+UT\mathbf{A}^+ = \mathbf{V}\mathbf{\Sigma}^{+}\mathbf{U}^T A+=VΣ+UT

Σ+\mathbf{\Sigma}^{+}Σ+ 的做法是把非零奇异值取倒数，零奇异值仍然保持为零。

10. CP Decomposition

CP，CANDECOMP/PARAFAC，分解的核心思想是：把一个张量近似表示成若干个 rank-one tensors 的和。

对三阶张量

X∈RI×J×K\mathcal{X}\in\mathbb{R}^{I\times J\times K} X∈RI×J×K

CP 分解写作：

X≈∑r=1Rar∘br∘cr\mathcal{X} \approx \sum_{r=1}^{R} \mathbf{a}_r\circ \mathbf{b}_r\circ \mathbf{c}_r X≈r=1∑R​ar​∘br​∘cr​

元素形式为：

xijk≈∑r=1Rairbjrckrx_{ijk} \approx \sum_{r=1}^{R} a_{ir}b_{jr}c_{kr} xijk​≈r=1∑R​air​bjr​ckr​

如果把所有 ar,br,cr\mathbf{a}_r,\mathbf{b}_r,\mathbf{c}_rar​,br​,cr​ 分别拼成矩阵：

A=[a1,…,aR],B=[b1,…,bR],C=[c1,…,cR]\mathbf{A}=[\mathbf{a}_1,\ldots,\mathbf{a}_R], \quad \mathbf{B}=[\mathbf{b}_1,\ldots,\mathbf{b}_R], \quad \mathbf{C}=[\mathbf{c}_1,\ldots,\mathbf{c}_R] A=[a1​,…,aR​],B=[b1​,…,bR​],C=[c1​,…,cR​]

可以简写为：

X≈[[A,B,C]]\mathcal{X}\approx[[\mathbf{A},\mathbf{B},\mathbf{C}]] X≈[[A,B,C]]

三阶张量的展开形式：

X(1)≈A(C⊙B)T\mathbf{X}_{(1)} \approx \mathbf{A}(\mathbf{C}\odot \mathbf{B})^T X(1)​≈A(C⊙B)T

X(2)≈B(C⊙A)T\mathbf{X}_{(2)} \approx \mathbf{B}(\mathbf{C}\odot \mathbf{A})^T X(2)​≈B(C⊙A)T

X(3)≈C(B⊙A)T\mathbf{X}_{(3)} \approx \mathbf{C}(\mathbf{B}\odot \mathbf{A})^T X(3)​≈C(B⊙A)T

CP-ALS 的常见求解思路是：固定其他因子矩阵，轮流更新某一个因子矩阵，把问题变成最小二乘问题。目标函数可以写成：

min⁡A(1),…,A(N)∥X−[[A(1),A(2),…,A(N)]]∥F2\min_{\mathbf{A}^{(1)},\ldots,\mathbf{A}^{(N)}} \left\| \mathcal{X} -[[\mathbf{A}^{(1)},\mathbf{A}^{(2)},\ldots,\mathbf{A}^{(N)}]] \right\|_F^2 A(1),…,A(N)min​∥∥∥∥​X−[[A(1),A(2),…,A(N)]]∥∥∥∥​F2​

CP 分解的 rank 是能表示该张量所需的最少 rank-one tensors 数量。

11. CP 分解的唯一性

矩阵的低秩分解通常不唯一。例如：

X=ABT=(AW)(BW−T)T\mathbf{X} = \mathbf{A}\mathbf{B}^T = (\mathbf{A}\mathbf{W})(\mathbf{B}\mathbf{W}^{-T})^T X=ABT=(AW)(BW−T)T

只要 W\mathbf{W}W 可逆，就能得到不同的因子。

张量 CP 分解在一定条件下反而可能唯一。三阶 CP 分解的一个经典充分条件是 Kruskal 条件：

kA+kB+kC≥2R+2k_{\mathbf{A}}+k_{\mathbf{B}}+k_{\mathbf{C}}\ge 2R+2 kA​+kB​+kC​≥2R+2

其中 kAk_{\mathbf{A}}kA​ 表示矩阵 A\mathbf{A}A 的 k-rank，也就是任意 kkk 列都线性无关时最大的 kkk。

CP 仍然存在两个不可避免的不确定性：

  • 列置换不确定性：因子矩阵的第 rrr 列可以整体换顺序。
  • 缩放不确定性：只要 αrβrγr=1\alpha_r\beta_r\gamma_r=1αr​βr​γr​=1，就有

ar∘br∘cr=(αrar)∘(βrbr)∘(γrcr)\mathbf{a}_r\circ\mathbf{b}_r\circ\mathbf{c}_r = (\alpha_r\mathbf{a}_r) \circ (\beta_r\mathbf{b}_r) \circ (\gamma_r\mathbf{c}_r) ar​∘br​∘cr​=(αr​ar​)∘(βr​br​)∘(γr​cr​)

12. Tucker Decomposition

Tucker 分解可以看作张量版 PCA。它把原张量分解成一个 core tensor 和每个 mode 上的 factor matrix。

对三阶张量：

X≈G×1A×2B×3C\mathcal{X} \approx \mathcal{G} \times_1 \mathbf{A} \times_2 \mathbf{B} \times_3 \mathbf{C} X≈G×1​A×2​B×3​C

也可以记作：

X≈[[G;A,B,C]]\mathcal{X} \approx [[\mathcal{G};\mathbf{A},\mathbf{B},\mathbf{C}]] X≈[[G;A,B,C]]

其中：

  • G\mathcal{G}G 是 core tensor。
  • A∈RI×P\mathbf{A}\in\mathbb{R}^{I\times P}A∈RI×P。
  • B∈RJ×Q\mathbf{B}\in\mathbb{R}^{J\times Q}B∈RJ×Q。
  • C∈RK×R\mathbf{C}\in\mathbb{R}^{K\times R}C∈RK×R。

元素形式为：

xijk≈∑p=1P∑q=1Q∑r=1Rgpqraipbjqckrx_{ijk} \approx \sum_{p=1}^{P} \sum_{q=1}^{Q} \sum_{r=1}^{R} g_{pqr}a_{ip}b_{jq}c_{kr} xijk​≈p=1∑P​q=1∑Q​r=1∑R​gpqr​aip​bjq​ckr​

CP 可以看作 Tucker 的特殊情况：core tensor 是超对角张量。

三阶 Tucker 的 matricized 形式：

X(1)≈AG(1)(C⊗B)T\mathbf{X}_{(1)} \approx \mathbf{A}\mathbf{G}_{(1)}(\mathbf{C}\otimes\mathbf{B})^T X(1)​≈AG(1)​(C⊗B)T

X(2)≈BG(2)(C⊗A)T\mathbf{X}_{(2)} \approx \mathbf{B}\mathbf{G}_{(2)}(\mathbf{C}\otimes\mathbf{A})^T X(2)​≈BG(2)​(C⊗A)T

X(3)≈CG(3)(B⊗A)T\mathbf{X}_{(3)} \approx \mathbf{C}\mathbf{G}_{(3)}(\mathbf{B}\otimes\mathbf{A})^T X(3)​≈CG(3)​(B⊗A)T

Tucker 分解通常不唯一，因为可以把一个可逆变换吸收到 core tensor，再用逆变换补偿 factor matrix：

[G;A,B,C]=[G×1U×2V×3W;AU−1,BV−1,CW−1][\mathcal{G};\mathbf{A},\mathbf{B},\mathbf{C}] = [\mathcal{G}\times_1\mathbf{U}\times_2\mathbf{V}\times_3\mathbf{W}; \mathbf{A}\mathbf{U}^{-1}, \mathbf{B}\mathbf{V}^{-1}, \mathbf{C}\mathbf{W}^{-1}] [G;A,B,C]=[G×1​U×2​V×3​W;AU−1,BV−1,CW−1]

常见计算方法：

  • HOSVD：对每个 mode 的 unfolding 做 SVD。
  • HOOI：在 HOSVD 初始化后，交替优化每个 factor matrix。

13. n-Rank

张量 X\mathcal{X}X 的 n-rank 指 mode-nnn unfolding 矩阵 X(n)\mathbf{X}_{(n)}X(n)​ 的列秩：

rankn(X)=rank(X(n))\mathrm{rank}_n(\mathcal{X})=\mathrm{rank}(\mathbf{X}_{(n)}) rankn​(X)=rank(X(n)​)

如果

Rn=rankn(X)R_n=\mathrm{rank}_n(\mathcal{X}) Rn​=rankn​(X)

那么 X\mathcal{X}X 可以说是一个 rank-(R1,R2,…,RN)(R_1,R_2,\ldots,R_N)(R1​,R2​,…,RN​) 张量。

这个概念在 Tucker 分解中尤其常用，因为 Tucker rank 本质上就是每个 unfolding 的矩阵秩组合。

14. 其他分解：INDSCAL、CANDELINC、PARAFAC2

INDSCAL 是 CP 的一个特殊情况，适用于在两个 mode 上对称的三阶张量。例如：

X∈RI×I×K\mathcal{X}\in\mathbb{R}^{I\times I\times K} X∈RI×I×K

并且 xijk=xjikx_{ijk}=x_{jik}xijk​=xjik​，则 INDSCAL 模型可写为：

X≈[[A,A,C]]=∑r=1Rar∘ar∘cr\mathcal{X} \approx [[\mathbf{A},\mathbf{A},\mathbf{C}]] = \sum_{r=1}^{R} \mathbf{a}_r\circ \mathbf{a}_r\circ \mathbf{c}_r X≈[[A,A,C]]=r=1∑R​ar​∘ar​∘cr​

CANDELINC 可以看作带线性约束的 CP 分解。普通 CP 是：

X≈[[A,B,C]]\mathcal{X}\approx[[\mathbf{A},\mathbf{B},\mathbf{C}]] X≈[[A,B,C]]

而 CANDELINC 写作：

X≈[[ΦAA^,ΦBB^,ΦCC^]]\mathcal{X} \approx [[\mathbf{\Phi}_A\hat{\mathbf{A}}, \mathbf{\Phi}_B\hat{\mathbf{B}}, \mathbf{\Phi}_C\hat{\mathbf{C}}]] X≈[[ΦA​A^,ΦB​B^,ΦC​C^]]

其中 ΦA,ΦB,ΦC\mathbf{\Phi}_A,\mathbf{\Phi}_B,\mathbf{\Phi}_CΦA​,ΦB​,ΦC​ 是约束矩阵或变换矩阵。当约束后维度更小，它可以减少计算量。

PARAFAC2 严格来说不是对一个固定尺寸三阶张量做分解，而是对一组矩阵做分解：

Xk∈RIk×J,k=1,…,K\mathbf{X}_k\in\mathbb{R}^{I_k\times J}, \quad k=1,\ldots,K Xk​∈RIk​×J,k=1,…,K

这里 IkI_kIk​ 可以随 kkk 改变。模型形式为：

Xk≈UkSkVT,k=1,…,K\mathbf{X}_k \approx \mathbf{U}_k \mathbf{S}_k \mathbf{V}^T, \quad k=1,\ldots,K Xk​≈Uk​Sk​VT,k=1,…,K

其中：

  • Uk∈RIk×R\mathbf{U}_k\in\mathbb{R}^{I_k\times R}Uk​∈RIk​×R
  • Sk∈RR×R\mathbf{S}_k\in\mathbb{R}^{R\times R}Sk​∈RR×R，通常是对角矩阵
  • V∈RJ×R\mathbf{V}\in\mathbb{R}^{J\times R}V∈RJ×R，在所有 kkk 上共享

为了增强唯一性，PARAFAC2 通常会加入约束：

UkTUk\mathbf{U}_k^T\mathbf{U}_k UkT​Uk​

在不同 kkk 上保持不变。

15. 总结

这篇笔记可以按下面这条线理解：

  1. 先掌握 tensor 的基本索引、fiber、slice、norm、inner product。
  2. 再理解 rank-one tensor，因为 CP 分解就是 rank-one tensor 的求和。
  3. Matricization 和 Khatri-Rao product 是 CP/ALS 公式的关键。
  4. Tucker 分解更像高阶 PCA：用 core tensor 表示不同 components 之间的交互。
  5. PARAFAC2 适合一组矩阵尺寸不完全一致，但仍希望共享某些潜在结构的场景。

从机器学习角度看，张量分解可以用于降维、缺失值补全、多视角数据建模、推荐系统、时序数据分析和多模态数据建模。后续如果要真正做实验，可以从 Python 的 tensorly 库开始，把 CP、Tucker、PARAFAC2 都跑一遍。
