AMA4850 最优化期末复习提纲：Convexity、Duality、KKT 与数值优化

这篇笔记根据 AMA4850 Final Exam Review 整理，主要覆盖凸性分析、LP/SDP 对偶、KKT 条件、最速下降、拟牛顿法、罚函数和 barrier method。原 PDF 更像考试复习提纲，这里整理成博客版 checklist，方便考前快速回忆。

1. 考试信息与复习重点

根据原复习材料：

  • 考试时间：2026 年 5 月 8 日，12:30 PM - 2:30 PM。
  • 考试形式：open-book，可使用纸质材料。
  • 电子设备不允许使用，基础计算器除外。
  • 带星号的章节是高优先级复习内容。

核心主题可以分成四组：

  1. 凸分析与矩阵基础：convexity、positive semidefinite matrix、Schur complement、Taylor theorem。
  2. 对偶理论：LP duality、SDP duality、strong duality。
  3. 无约束优化：stationary point、gradient descent、Armijo rule、quasi-Newton。
  4. 约束优化：KKT、MFCQ、Slater condition、penalty/barrier methods。

2. 凸性分析

判断函数是否 convex 是复习材料的第一重点。

对 f:Rn→Rf:\mathbb{R}^n\to\mathbb{R}f:Rn→R：

  • 如果 fff 二阶可微，则

f convex⟺∇2f(x)⪰0, ∀x∈dom(f)f \text{ convex} \quad \Longleftrightarrow \quad \nabla^2 f(x)\succeq 0,\ \forall x\in \mathrm{dom}(f) f convex⟺∇2f(x)⪰0, ∀x∈dom(f)

  • 如果 fff 一阶可微，则 convex 等价于一阶条件：

f(y)≥f(x)+∇f(x)T(y−x),∀x,yf(y)\ge f(x)+\nabla f(x)^T(y-x),\quad \forall x,y f(y)≥f(x)+∇f(x)T(y−x),∀x,y

  • 如果 fff 不可微，可以从 epigraph 判断：

epi(f)={(x,t):f(x)≤t}\mathrm{epi}(f)=\{(x,t): f(x)\le t\} epi(f)={(x,t):f(x)≤t}

如果 epi(f)\mathrm{epi}(f)epi(f) 是凸集，则 fff 是凸函数。

常用判断路线：

  1. 先看定义域是否 convex。
  2. 如果可求 Hessian，检查 Hessian 是否 PSD。
  3. 如果函数由已知 convex 函数组合而成，用 composition rule。
  4. 如果是 norm、max、log-sum-exp、quadratic form，优先套常见结论。

3. Stationary Point 与二阶分类

对无约束优化问题：

min⁡xf(x)\min_x f(x) xmin​f(x)

如果 x^\* 是局部极小点，并且 fff 可微，则必要条件是：

\nabla f(x^\*)=0

这样的点叫 stationary point。

如果 fff 二阶可微，可以用 Hessian 分类：

  • \nabla^2 f(x^\*)\succ 0：strict local minimizer。
  • \nabla^2 f(x^\*)\prec 0：strict local maximizer。
  • Hessian indefinite：saddle point。
  • Hessian semidefinite：需要更高阶信息或其他分析。

Saddle point 的直观含义是：梯度为 0，但某些方向上函数值增加，另一些方向上函数值减少。

4. Positive Semidefinite Matrix

PSD 矩阵在 convexity、SDP、KKT 和 Schur complement 里反复出现。

对称矩阵 A∈SnA\in S^nA∈Sn：

A⪰0⟺xTAx≥0, ∀xA\succeq 0 \quad \Longleftrightarrow \quad x^TAx\ge 0,\ \forall x A⪰0⟺xTAx≥0, ∀x

常用判断方法：

  • 所有特征值非负。
  • 所有 principal minors 非负。
  • 如果 A≻0A\succ 0A≻0，可用 Sylvester criterion：所有 leading principal minors 都大于 0。

对 quadratic form：

q(x)=xTAxq(x)=x^TAx q(x)=xTAx

如果 A⪰0A\succeq 0A⪰0，则 q(x)q(x)q(x) convex；如果 A≻0A\succ 0A≻0，则 strictly convex。

5. Schur Complement

Schur complement 是 SDP reformulation 的核心工具。

设

M=[ABBTC]M= \begin{bmatrix} A & B\\ B^T & C \end{bmatrix} M=[ABT​BC​]

如果 C≻0C\succ 0C≻0，则：

M⪰0⟺A−BC−1BT⪰0M\succeq 0 \quad \Longleftrightarrow \quad A-BC^{-1}B^T\succeq 0 M⪰0⟺A−BC−1BT⪰0

如果 A≻0A\succ 0A≻0，则：

M⪰0⟺C−BTA−1B⪰0M\succeq 0 \quad \Longleftrightarrow \quad C-B^TA^{-1}B\succeq 0 M⪰0⟺C−BTA−1B⪰0

考试中常见用法是把非线性约束改写成 LMI。例如：

t≥xTP−1x,P≻0t\ge x^TP^{-1}x,\quad P\succ 0 t≥xTP−1x,P≻0

可改写为：

[txTxP]⪰0\begin{bmatrix} t & x^T\\ x & P \end{bmatrix} \succeq 0 [tx​xTP​]⪰0

6. LP Duality

线性规划常见 primal 形式：

min⁡xcTxs.t. Ax=b, x≥0\min_x c^Tx \quad \text{s.t. } Ax=b,\ x\ge 0 xmin​cTxs.t. Ax=b, x≥0

对应 dual：

max⁡ybTys.t. ATy≤c\max_y b^Ty \quad \text{s.t. } A^Ty\le c ymax​bTys.t. ATy≤c

复习时重点不是死记某一种形式，而是掌握构造逻辑：

  1. 把 primal 写成标准形式。
  2. 为约束引入 Lagrange multiplier。
  3. 对 xxx 取 infimum，得到 dual function。
  4. 写出 dual feasible condition。
  5. 比较 weak duality / strong duality。

Weak duality：

d^\* \le p^\*

Strong duality 在满足适当条件时成立：

d^\*=p^\*

7. SDP Duality

SDP 可以看作 LP 的矩阵推广，变量中出现 PSD 约束。

一种常见 primal 形式：

min⁡XTr(CX)\min_X \mathrm{Tr}(CX) Xmin​Tr(CX)

subject to：

Tr(AiX)=bi,i=1,…,m\mathrm{Tr}(A_iX)=b_i,\quad i=1,\ldots,m Tr(Ai​X)=bi​,i=1,…,m

X⪰0X\succeq 0 X⪰0

对应 dual：

max⁡ybTy\max_y b^Ty ymax​bTy

subject to：

C−∑i=1myiAi⪰0C-\sum_{i=1}^{m}y_iA_i\succeq 0 C−i=1∑m​yi​Ai​⪰0

SDP 题目常考：

  • 如何把问题改写成 PSD matrix constraint。
  • 如何从 primal 写 dual。
  • 如何用 primal-dual relationship 验证 optimality。
  • Slater condition 在 strong duality 中的作用。

8. Gradient Descent 与 Steepest Descent

无约束优化：

min⁡xf(x)\min_x f(x) xmin​f(x)

Gradient descent 更新：

xk+1=xk−αk∇f(xk)x_{k+1}=x_k-\alpha_k\nabla f(x_k) xk+1​=xk​−αk​∇f(xk​)

如果使用 exact line search：

αk=arg⁡min⁡α>0f(xk−α∇f(xk))\alpha_k = \arg\min_{\alpha>0} f(x_k-\alpha\nabla f(x_k)) αk​=argα>0min​f(xk​−α∇f(xk​))

如果使用 constant stepsize，需要关注收敛条件。对 LLL-smooth 函数，常见条件是：

如果 fff convex 且 smooth，可以得到函数值下降；如果 strong convex，还能得到更强的收敛结论。

9. Armijo Rule

Armijo rule 用来做 backtracking line search。给定下降方向 dkd_kdk​，希望找到 αk\alpha_kαk​ 满足：

f(xk+αkdk)≤f(xk)+cαk∇f(xk)Tdkf(x_k+\alpha_k d_k) \le f(x_k)+c\alpha_k\nabla f(x_k)^Td_k f(xk​+αk​dk​)≤f(xk​)+cαk​∇f(xk​)Tdk​

其中 c∈(0,1)c\in(0,1)c∈(0,1)。

典型算法：

  1. 初始化 α=1\alpha=1α=1。
  2. 如果 Armijo 条件不满足，令 α←βα\alpha\leftarrow \beta\alphaα←βα，其中 β∈(0,1)\beta\in(0,1)β∈(0,1)。
  3. 重复直到条件满足。

重点理解：Armijo rule 不追求最优步长，而是找一个足够下降且计算便宜的步长。

10. Quasi-Newton 与 BFGS

Newton method 使用 Hessian：

xk+1=xk−[∇2f(xk)]−1∇f(xk)x_{k+1} = x_k -[\nabla^2 f(x_k)]^{-1}\nabla f(x_k) xk+1​=xk​−[∇2f(xk​)]−1∇f(xk​)

但 Hessian 计算和求逆可能很贵，所以 quasi-Newton 用近似矩阵代替 Hessian 或 inverse Hessian。

BFGS 的核心是用迭代信息更新 Hessian 近似。定义：

Bk+1sk=ykB_{k+1}s_k=y_k Bk+1​sk​=yk​

  • 会执行多步 quasi-Newton 更新。
  • 理解 sks_ksk​ 和 yky_kyk​ 的含义。
  • 知道 BFGS 保持正定性的条件通常需要 skTyk>0s_k^Ty_k>0skT​yk​>0。

11. KKT Conditions

考虑约束优化：

min⁡xf(x)\min_x f(x) xmin​f(x)

subject to：

gi(x)≤0,i=1,…,mg_i(x)\le 0,\quad i=1,\ldots,m gi​(x)≤0,i=1,…,m

hj(x)=0,j=1,…,ph_j(x)=0,\quad j=1,\ldots,p hj​(x)=0,j=1,…,p

Lagrangian：

L(x,λ,ν)=f(x)+∑i=1mλigi(x)+∑j=1pνjhj(x)L(x,\lambda,\nu) = f(x)+\sum_{i=1}^{m}\lambda_i g_i(x)+\sum_{j=1}^{p}\nu_j h_j(x) L(x,λ,ν)=f(x)+i=1∑m​λi​gi​(x)+j=1∑p​νj​hj​(x)

KKT 条件包括：

Primal feasibility

g_i(x^\*)\le 0,\quad h_j(x^\*)=0

Dual feasibility

\lambda_i^\*\ge 0

Complementary slackness

\lambda_i^\*g_i(x^\*)=0

Stationarity

\nabla_x L(x^\*,\lambda^\*,\nu^\*)=0

KKT 是考试核心。要特别注意：

  • 对一般非凸问题，KKT 通常只是 necessary condition，需要 constraint qualification。
  • 对 convex optimization，如果满足适当条件，KKT 可以是 necessary and sufficient。
  • MFCQ、Slater condition 是常见 constraint qualification。

12. MFCQ 与 Slater Condition

MFCQ，Mangasarian-Fromovitz constraint qualification，通常用于保证 KKT 必要性。

Slater condition 常用于凸优化 strong duality。对 convex problem，如果存在严格可行点：

gi(x)<0,hj(x)=0g_i(x)<0,\quad h_j(x)=0 gi​(x)<0,hj​(x)=0

则 strong duality 通常成立，并且 KKT 条件表现良好。

复习时可以这样记：

  • MFCQ：偏一般约束优化，服务于 KKT necessary condition。
  • Slater：偏 convex optimization，服务于 strong duality 和 KKT。

13. Penalty Method

Penalty method 把约束问题转成无约束问题。例如等式约束：

h(x)=0h(x)=0 h(x)=0

可以构造：

min⁡xf(x)+ρ2∥h(x)∥2\min_x f(x)+\frac{\rho}{2}\|h(x)\|^2 xmin​f(x)+2ρ​∥h(x)∥2

其中 ρ\rhoρ 是 penalty parameter。随着 ρ→∞\rho\to\inftyρ→∞，解会更接近可行域。

优点：形式简单，可以用无约束优化方法。

缺点：ρ\rhoρ 很大时问题会变得 ill-conditioned，数值上更难求。

14. Barrier Method

Barrier method 主要用于不等式约束：

gi(x)<0g_i(x)<0 gi​(x)<0

常见 logarithmic barrier：

ϕ(x)=−∑i=1mlog⁡(−gi(x))\phi(x) = -\sum_{i=1}^{m}\log(-g_i(x)) ϕ(x)=−i=1∑m​log(−gi​(x))

构造问题：

min⁡xtf(x)+ϕ(x)\min_x tf(x)+\phi(x) xmin​tf(x)+ϕ(x)

或者：

min⁡xf(x)+1tϕ(x)\min_x f(x)+\frac{1}{t}\phi(x) xmin​f(x)+t1​ϕ(x)

当 xxx 靠近约束边界时，barrier term 会趋向无穷大，从而阻止迭代点离开可行域。

复习重点：

  • penalty method 允许从不可行点逼近可行性。
  • barrier method 通常要求从严格可行点开始。
  • penalty parameter 和 barrier parameter 的变化方向要分清。

15. 考前 Checklist

考前可以按这个顺序复习：

  1. 会用 Hessian 判断 convexity。
  2. 会判断 PSD/PD，熟悉 Sylvester criterion。
  3. 会用 Schur complement 改写 SDP 约束。
  4. 会从 LP/SDP primal 写 dual。
  5. 会求 stationary point，并用 Hessian 分类。
  6. 会写 gradient descent、Armijo rule、Newton/quasi-Newton 更新。
  7. 会完整写 KKT 四组条件。
  8. 会说明 MFCQ 和 Slater condition 的作用。
  9. 会构造 quadratic penalty 和 log barrier。

如果时间很紧，优先复习星号主题：convexity、LP/SDP duality、stationary point、gradient descent convergence、quasi-Newton、KKT、penalty/barrier。
