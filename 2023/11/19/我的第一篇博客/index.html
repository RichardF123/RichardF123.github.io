我的第一篇博客

前记

想搭建一个自己的博客，其实已经是很久以前的想法了。

平时学习的东西很多，遇到一些有意思的问题、方法或者想法时，总希望能把它们记录下来。相比手写笔记，博客更容易整理，也更方便长期回看。

为什么写博客

写博客对我来说有几个意义：

  1. 记录学习过程。
  2. 强迫自己把知识讲清楚。
  3. 给未来的自己留下复盘材料。
  4. 如果能帮助到别人，那就更好了。

很多时候，一个知识点看似学会了，但真正写出来才会发现自己哪里还不清楚。

博客会写什么

之后我希望主要记录这些内容：

  • 编程学习笔记。
  • 数据科学和机器学习。
  • 数学课程总结。
  • 项目复盘。
  • 一些个人想法。

现在的内容可能还很粗糙，但博客本来就是慢慢积累的地方。

搭建过程的感受

搭建博客比想象中麻烦。

主题配置、部署、GitHub Pages、Markdown、图片路径、编码问题，每一个细节都可能卡住。

但也正因为这些问题，整个过程让我更熟悉了：

  • Git 和 GitHub。
  • Hexo 的构建流程。
  • 静态网站部署。
  • Markdown 写作。

后记

希望这个博客能一直更新下去。

不一定每篇文章都很完整，但只要持续记录，时间久了就会形成自己的知识轨迹。

这篇文章就当作一个开始。
